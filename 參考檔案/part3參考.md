所有 Repositories（含私有）

是否有 重要 Repo 要備份

是否有 GitHub Copilot / Education 綁定

是否有 組織（Organization）管理權

👉 若有重要內容，請先：

git clone 備份

或轉移 Repo 到其他帳號 / 組織
