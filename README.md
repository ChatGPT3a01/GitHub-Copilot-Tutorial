## 課程簡介

本教學介紹 GitHub 與 OpenAI 合作開發的 **GitHub Copilot** AI 程式碼助手。GitHub Copilot 是目前最廣泛使用的 AI 編程工具，能夠根據上下文自動補全程式碼、生成函式、撰寫測試，大幅提升開發效率。

## GitHub Copilot 核心功能

| 功能 | 說明 |
|------|------|
| 自動補全 | 根據上下文預測並補全程式碼 |
| Copilot Chat | AI 對話式程式碼助手 |
| 註解驅動 | 用自然語言描述需求生成程式碼 |
| 單元測試 | 自動生成測試案例 |
| 程式碼重構 | 優化現有程式碼結構 |
| 文件生成 | 自動撰寫程式碼註解與文件 |
