# BMI 計算器說明文件

## 概述
此 Python 腳本 ([`practice6.py`](practice6.py )) 實現了一個身體質量指數 (BMI) 計算器，它接受用戶輸入的體重和身高，計算 BMI，並根據標準 BMI 類別提供健康建議。

## 功能
- 從用戶提供的體重 (kg) 和身高 (cm) 計算 BMI
- 根據 BMI 範圍提供健康建議
- 包括正數值的輸入驗證
- 優雅處理無效輸入並顯示錯誤訊息

## 使用方法
在 Python 環境中運行腳本。它會提示輸入體重和身高。

```python
# 範例執行：
# 請輸入體重 (公斤): 70
# 請輸入身高 (公分): 175
# 你的 BMI 值為: 22.86
# 體重正常，請保持健康的生活方式。
```

## 函數詳情
- [`get_bmi_advice(bmi)`](practice6.py ): 根據 BMI 值使用預定義類別返回健康建議字符串。

## 依賴項
- Python 3.x (僅使用內建函數)

## 注意事項
- 身高在內部從公分轉換為公尺。
- BMI 類別遵循標準醫療指南。
- 對於關鍵健康應用，請始終驗證計算結果。
